# 🎮 Number Guessing Game

This is a simple Python number guessing game.  
The computer selects a random number between 1 and 100.  
Players have to guess the correct number.  
The game gives hints if the guess is too high or too low.  
It keeps counting the number of attempts.  
The game ends when the player guesses correctly.  
Built using Python and the random module.  
Great project for Python beginners.  
Run the game using: `python game.py`
